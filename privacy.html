Kebijakan Privasi

Terakhir Diperbarui: 23 Juni 2026

Monsterdex Community ("Aplikasi") berkomitmen untuk melindungi privasi data pribadi Anda selaku pengguna. Kebijakan Privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi informasi Anda.

1. Informasi yang Kami Kumpulkan

Aplikasi ini mengumpulkan beberapa informasi pribadi Anda ketika Anda mendaftar atau mengedit profil:

  • Nama & Email: Digunakan untuk registrasi dan otentikasi masuk ke akun Anda.
  • Password: Disimpan dengan aman menggunakan enkripsi (hashing) dan hanya digunakan untuk verifikasi login.
  • Foto Profil (Avatar): Foto yang Anda unggah secara sukarela untuk profil Anda. Foto ini diunggah dan disimpan secara aman di cloud (Cloudinary).
  • Biodata (Bio): Informasi opsional tentang diri Anda yang dapat Anda edit di profil.

2. Data Pihak Ketiga & Integrasi API

Aplikasi ini menggunakan data publik dari PokeAPI (pokeapi.co) untuk menampilkan informasi monster (karakter). Karakter, nama, gambar, dan merek dagang yang berhubungan dengan monster tersebut adalah milik sah dari pemegang hak cipta masing-masing (Nintendo, Game Freak, The Pokémon Company). Kami tidak mengumpulkan data pribadi apa pun melalui PokeAPI.

3. Bagaimana Kami Menggunakan Informasi Anda

Informasi yang kami kumpulkan hanya digunakan untuk keperluan fungsionalitas di dalam aplikasi, seperti:

  • Mengelola akun pengguna Anda (Otentikasi).
  • Menampilkan profil Anda kepada Anda di aplikasi.
  • Menyimpan preferensi tim monster dan favorit Anda yang terhubung dengan akun Anda.

4. Keamanan Data

Semua lalu lintas data antara aplikasi dan server kami menggunakan enkripsi SSL/HTTPS. Kredensial password disimpan menggunakan enkripsi hash bcrypt di database MySQL aman kami. Namun, tidak ada transmisi data melalui Internet yang 100% aman, sehingga kami tidak dapat menjamin keamanan mutlak data Anda.

5. Perubahan Kebijakan Privasi

Kami dapat memperbarui Kebijakan Privasi ini dari waktu ke waktu. Setiap perubahan akan diberitahukan dengan memperbarui tanggal "Terakhir Diperbarui" di bagian atas halaman ini.
